Le concours 2010 a démarré et nous n'avons plus besoin de site web pour cette année. Contactez-nous si vous souhaitez refaire un concours avec votre site

Vous avez un site ? Nous avons besoin de vous !

Nous sommes un groupe de professionnels de l'optimisation des performances web. Nous organisons ce concours d'optimisation avec l'aide de nos sponsors YOTTAA et Alwaysdata.

Un concours d'optimisation des performances web ?

Lorsque le concours débutera le 25 octobre, nous donnerons aux participants un fichier ZIP contenant une page web complète.

Les participants feront de leur mieux pour l'optimiser et enverront leurs modifications sur les serveurs fournis par Alwaysdata.

A la fin du concours (le 30 novembre), nous étudierons les résultats et les modifications de chaque participant pour décider du vainqueur.

Pourquoi avons-nous besoin de votre site

Afin d'avoir un concours vraiment intéressant, nous recherchons des sites du "monde réel" à optimiser

Nous avons déjà quelques candidats mais nous cherchons encore, en particulier si vous avez énormément de traffic, que vous avez beaucoup de CSS ou de Javascript, et bien sur que vous voulez accélérer le rendu !

Vous possédez ou vous travaillez pour un tel site ? Ceci vous intéressera :

Obtenez gratuitement pour votre site web :

  • Les meilleurs avis de plusieurs professionnels pour accélérer l'affichage de votre site
  • Des exemples complets et commentés avec la description du "comment", pas simplement une analyse sur le temps de chargement de votre site
  • Savez vous combien ce type d'ifnormation coûte aujourd'hui ? beaucoup ! Vous pouvez l'avoir gratuitement

Quels sont les risques ?

Aucun !

Mais ... tout le monde va voir mon code source, c'est dangereux non ?

Pas du tout! Nous allons donner aux participants des fichiers statiques, nous n'avons pas besoin de votre code serveur, simplement ce qui est déjà visible sur votre site.

Vous n'avez même pas à vous charger de nous donner une version statique de votre page, nous nous en chargeons.
