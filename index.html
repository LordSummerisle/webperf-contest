Brief

Zeroload is organizing the first web performance contest, with the support of YOTTAA and Alwaysdata.

We are a group of web performance junkies organizing a contest like the js1k competition but this time on the subject of WPO (Web Performance Optimization).

This is the first time a competition of this kind is organized (well, almost).

The contest will start started the 25th October and end the 30th November.

How to enter the competition ? (soon)

Download the files and register. The Zip contains a complete static webpage (.html) with all the required assets (js, css, backgrounds, ads) for the page to work.

Download the files and register. The Zip contains a complete static webpage (.html) with all the required assets (js, css, backgrounds, ads) for the page to work.

You will then receive a temporary FTP account on a French host (provided by our sponsor, Alwaysdata) to put your files.

You will also be given a uniqid wich will be the directory name where to put your files.

All participant's directories are available on 3 different domains (entries.webperf-contest.com, s1.webperf-contest.com and s2.webperf-contest.com).

If you do not want to be copied, do not share your uniqid ! We won't be responsible for that.

How to win ?

When the contest ends, each participant's url will be tested in multiple tools and ranked by these categories :

  • Best YOTTAA score
  • Best onload & start render times
  • Smallest number of requests & total size

Measurements tools :

  • Yottaa
  • WPT paris location, DSL (1.5Mb/s, 50ms RTT)

All the participant's files will be hosted on the same location (alwaysdata servers).

You will be able to test your changes in the exact same conditions as us with, the FTP + HTTP account which will be given to you.
