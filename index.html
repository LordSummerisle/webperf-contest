Brief

Zeroload is organizing the first web performance contest, sponsored by YOTTAA.

We are a group of web performance junkies. We are planning a contest like the js1k competition but this time on the subject of WPO (Web Performance Optimization).

This is the first time a competition of this kind is organized (well, almost).

The contest will start the 25th October and end the 30th November.

How to enter the competition ?

When the contest starts, people will be able to download a Zip file from this website. The Zip will contain a complete static webpage (.html) with all the required assets (js, css, backgrounds, ads) for the page to work.

Each participants will receive a FTP account to put their files. Of course you won't be able to see other participant's urls before the end of the contest (also this removes ability to cheat with other people's optimized assets). The FTP accounts will be active for the time of the contest. Participants will have 3 different subdomains, to maximize possibilities. Please ask if you need something more.

Each participant will provide their best magic skills to make the example webpage fast.

How to win ?

When the contest ends, each participant's url will be tested in multiple tools and ranked by theses categories :

  • Best YOTTAA score
  • Best sum of YSLOW, Pagespeed & Zoompf scores
  • Shortest loading time (start render + onload)
  • Best progressive rendering (humain eye check)
  • Smallest website size, tiniest number of requests
  • Innovation award / best idea

Both first and second access to the page will be taken into account.

All measures will be done from a typical DSL connection (1.5Mb/s, 50ms RTT), on the same server, with online analysis tools like yottaa.com, webpagetest.org (Paris location) and zoompf.com. You will be able to test your changes in the exact same conditions as us with, the FTP + HTTP account which will be given to you.

Before the 30th november, provide us :
  1. an exhaustive list of your modifications. We will partly judge you on the modifications you brought, and one of our goal is to educate the community, so list them all and be precise if you do something unusual
  2. your name, email and website if any

What to win ?

  • An IPAD, generously given by our sponsor, YOTTAA
  • Fame!
  • Performance skills
  • Self accomplishement
  • Want to give something ? A book, ticket for a conference, license for your software/solution ?

Rules

  • No cheating ! Displaying a big JPG for the complete website is cheating (even if you strip exif data, yes!)
  • No server side code (it is disabled)
  • User experience should not be degraded (but you are welcome to improve it), which is to say, for example:
    • Once page is loaded, all original features must still work
    • Ads, widgets or trackers must still be visible and usable
    • Accessibility must be preserved (also you can improve it)
    • SEO must be preserved (also you can improve it)

Then, you can use whatever technique you want : data uris, mhtml (external, inline), script defer ...

You may deliver a page wich is not an exact pixel per pixel copy and do some arrangements (like replacing an image by some css or js code wich has a similar result), provided you respect the "user experience" rule. The judges will have the final word on what they think is acceptable or not (we will probably be less strict for the "innovation award / best idea" criteria. If a typical product owner would refuse your change, you may be outside the rules. Ask if you're really unsure.

Sponsors

YOTTAA is our main sponsor. YOTTAA is providing us web performance measurements for the participants entries. Also YOTTAA is giving an IPAD to the participant that will have the best YOTTAA score.

Do you want to help us and gain some buzz on your products or your event ? Contact us ! We welcome a licence of your product, a ticket to your event or some goodie at your name as additionnal prizes.

We already have some candidates but if you would like a free performance rework of your pages with some buzz around your website, we may use your pages as subject for the contest. Contact us !
