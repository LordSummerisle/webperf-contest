1/ Téléchargez la page web

⇩ téléchargez la page web

Dans ce fichier compressé se trouve l'ensemble des ressources nécessaires pour participer au concours : les fichiers html, css, js et les images.

2/ Inscrivez vous

Avec l'inscription, vous obtiendrez un identifiant unique qu'il vous faudra bien sur tenir secret. Non seulement cela sera votre accès FTP mais cela nous permettra de vous identifier.

Requis Je confirme être humain (merci!)
Optionnel Non, je ne suis pas un robot venu vous spamer

3/ Créez un gist privée

Un gist est une sorte de post-it en ligne que vous pouvez partagez.

Vous devez :

  1. créer un Gist privé (aucun enregistrement requis)
  2. Nommez le de cette façon : webperfcontest2010-nom
  3. Cliquez sur "create private gist"
  4. Copiez collez l'url générée dans le formulaire d'inscription

Pendant le concours, prenez bien note de toutes vos modifications dans ce Gist.

Cette liste de modifications aidera les juges :

  • à départager les candidats avec des temps de chargement/rendu ou des poids très proches (moins de 30ms, moins de 5ko)
  • à repérer d'éventuels tricheurs qui n'auraient fait que du copier/coller

Gist est un service gratuit de Github. Les Gists permettent de partager facilement du code ou du texte en ligne. Il n'est pas nécessaire d'être inscrit à Github pour utiliser ce service.

4/ Faites de votre mieux

Vous recevrez votre compte FTP ce soir ou demain. D'ici là, vous pouvez déjà commencer à modifier les fichiers en les testant de votre côté sur votre machine de développement.

Souvenez vous que l'hébergement est en France et que les temps seront constatés depuis la France. Il est donc valable de développer sur sa propre machine puis d'envoyer les modifications par FTP pour test lorsque vous avez quelque chose d'intéressant.

Enfin, si vous modifiez le HTML, le CSS ou le JavaScript, fournissez nous les sources modifiées avant minification, afin que nous puissions départager les meilleurs : créez un répertoire "source" et mettez y ces fichiers, cela sera suffisant.
