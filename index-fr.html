En bref

Zeroload organise le premier concours d'optimisations des performances web, avec le support de YOTTAA et d'Alwaysdata.

Nous sommes un groupe de professionnels de l'optimisation des performances web. Nous organisons un concours entre développeurs sur le thème des WPO (Web Performance Optimization).

C'est la première compétition de ce genre (ou presque).

Le concours commencera le 25 octobre et finira le 30 novembre.

Comment entrer dans la compétition ? (à venir)

Téléchargez le fichier et enregistrez vous. Le fichier est un ZIP contenant une page statique complète et tous les fichiers dépendants (JS, CSS, images, publicités ...).

Vous recevez ensuite un compte FTP temporaire chez un hébergeur français (fourni par notre sponsor, Alwaysdata) afin d'y déposer vos modifications.

Nous vous donnons également un identifiant unique qui sera le nom du répertoire dans lequel vos fichiers seront déposés. Ne le partagez pas avec autrui si vous ne voulez pas être copié !

Les répertoires des participants seront disponibles dans 3 domaines différents (entries.webperf-contest.com, s1.webperf-contest.com and s2.webperf-contest.com).

Comment gagner ?

A la fin du concours, nous testerons les URL de chaque participant pour déterminer le vainqueur pour ces catégories :

  • Meilleur score YOTTAA
  • Meilleurs temps onload et premier rendu
  • Plus petit poids et nombre de requêtes

Outils de mesure :

  • Yottaa
  • WPT à Paris, DSL (1.5Mb/s, 50ms RTT)

Tous les participants seront hébergés sur les mêmes serveurs.

Vous pourrez tester vous même votre performance dans exactement les mêmes conditions que les juges
